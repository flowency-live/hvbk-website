Accessibility Statement

Our commitment to making this website accessible to everyone.

Last updated: 20 June 2026

The Hi Vis Bookkeeper is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Our Commitment

We believe that the web should be accessible to everyone, regardless of ability. We have designed this website with accessibility in mind and strive to meet WCAG 2.1 Level AA standards.

👁

Visual

High contrast colours, resizable text, and clear typography for improved readability.

⌨

Keyboard

Full keyboard navigation support. All interactive elements are accessible without a mouse.

🗣

Screen Readers

Semantic HTML structure, ARIA labels, and descriptive alt text for assistive technologies.

📱

Responsive

Mobile-friendly design that works across all devices and screen sizes.

Accessibility Features

This website includes the following accessibility features:

Navigation

  • Clear, consistent navigation structure
  • Skip-to-content links for keyboard users
  • Logical heading hierarchy (H1, H2, H3)
  • Descriptive link text that makes sense out of context

Visual Design

  • Colour contrast ratios meeting WCAG 2.1 AA standards (minimum 4.5:1 for text)
  • Text resizable up to 200% without loss of content or functionality
  • No content relies solely on colour to convey meaning
  • Focus indicators visible on all interactive elements

Content

  • Alternative text for all meaningful images
  • Form fields with visible labels and error messages
  • Plain language avoiding jargon where possible
  • Tables with appropriate headers and structure

Technical

  • Valid, semantic HTML5 markup
  • ARIA attributes where appropriate
  • No automatic audio or video
  • Pages work without JavaScript for core content

Known Limitations

While we strive for full accessibility, we acknowledge some current limitations:

  • PDF documents: Some downloadable documents may not be fully accessible. We can provide alternative formats upon request.
  • Third-party content: Some embedded content (e.g., booking calendars) may have accessibility limitations beyond our control.
  • Older browsers: Some features may not work optimally in older browsers. We recommend using an up-to-date browser.

Browser and Assistive Technology

This website has been tested with the following combinations:

  • Chrome + NVDA (Windows)
  • Firefox + NVDA (Windows)
  • Safari + VoiceOver (macOS/iOS)
  • Edge + Narrator (Windows)

We aim to support the current and previous two versions of major browsers (Chrome, Firefox, Safari, Edge).

How to Adjust Your Settings

You can customise how you view this website using your browser or device settings:

Increase text size

  • Windows: Press Ctrl and + (plus) to zoom in, Ctrl and - (minus) to zoom out
  • Mac: Press Cmd and + (plus) to zoom in, Cmd and - (minus) to zoom out
  • Mobile: Use your device's accessibility settings to increase text size

High contrast mode

  • Windows: Settings > Accessibility > Contrast themes
  • Mac: System Preferences > Accessibility > Display > Increase contrast

Feedback and Contact

We welcome your feedback on the accessibility of this website. If you encounter any accessibility barriers or have suggestions for improvement, please contact us:

Email: hello@hivisbooks.co.uk
